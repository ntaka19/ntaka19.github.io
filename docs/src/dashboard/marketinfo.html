主要変動要因

  1. 再生可能エネルギーの供給量
  2. 風力・太陽光が強いと火力発電需要が減り、スポット価格を抑えます。

  4. 猛暑・厳冬は電力需要を急増させます。

  • 日本・欧州・北米とも、気温要因が最重要。
  • とくに夏入り前は、
  • 冷房需要の立ち上がり
  • 発電所の定期点検明け
  • 燃料在庫の積み増し
    が価格変動要因になります。
