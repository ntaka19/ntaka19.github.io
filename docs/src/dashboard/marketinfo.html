2025年8月3日時点の世界経済のニュースでは、米国経済は不安定な状況が続いており、7月の米雇用統計は市場予想を下回る伸びにとどまったことが大きく報じられています。具体的には、7月の非農業部門就業者数は7万3000人増で予想の約10万人増を下回り、失業率は4.2%へ悪化しました。その一方で平均時給は前年同月比3.9%増となりインフレ圧力はやや強いです。この雇用統計の結果により米連邦準備制度理事会（FRB）が近く利下げに踏み切る可能性が高まったことや、米中貿易摩擦激化の懸念もあって安全資産である金の価格が上昇しています[1][3][4]。また、中央銀行は金融政策を据え置いているものの、貿易協議の進展と新たな関税措置が市場に影響を与えています[2]。

顕著な株価の増減としては、米雇用統計の悪化を受けてダウ平均株価が急落したことが挙げられます[3]。米国を中心に労働市場の減速が警戒され、世界の主要株式市場に波及効果を与えている可能性があります。日産自動車の2025年4～6月期決算は1157億円の利益を計上し、好調な企業決算も一部で市場の反応要因となっています[3][2]。

USD/JPY（ドル円）については、米国の弱い雇用統計を受けてドルが売られ、円が買われ一時1ドル＝147円台後半まで円高が進行しました。この動きは、FRBの利下げ期待とともに米ドルの魅力低下を映しており、円が安全通貨として注目されています[3][1]。
