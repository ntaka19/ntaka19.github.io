2. LNG（液化天然ガス）市場

  • 価格動向: アジアのJKM（日本市場価格）は欧州のTTFより高値で推移しており、日本・東アジア地域の冬季需要が先行して価格を押し上げている。TTFは欧州の備蓄状況が比較的充足しているためJKMほどの上昇にはなっていない[3]。
  • 変動要因: 欧州は冬季に向けて備蓄水準を高く維持しており、供給の安定化が認められる。一方、米国の輸出施設容量は増強されており輸出余力が拡大傾向。地域的に需給が異なるため価格差が顕著。冬季の需要先取りも価格を押し上げる要因であり、今後は天候などの不確実性に左右されやすい。

3. 電力市場

  • 価格動向: 日本のJEPX市場は気温の変動に左右されやすく、直近では秋口の気温低下で暖房需要増を受けて若干の価格上昇がみられる。欧州は再生可能エネルギー供給の増加が進む一方、燃料価格の影響を受けて価格に高変動性が残る。北米は州ごとの差異が大きく、特にLNG・石炭価格動向が発電コストに直接影響している[3]。
  • 変動要因: 再生可能エネルギーの導入増加が長期的な価格安定化を後押しするが、短期的には気温変動による需要の変動や発電燃料価格の影響大きく、需給バランスが乱高下している。
