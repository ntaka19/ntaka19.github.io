2025年7月31日時点の世界経済ニュースでは、IMF（国際通貨基金）が2025年の世界経済成長率予測を3.0％に上方修正しました。これは4月時点から0.2ポイントの引き上げで、5月の米中関税緩和やトランプ前大統領による関税引き上げ幅の縮小が背景にあります。米国の成長率予測は0.9％、日本は0.7％にそれぞれ0.1ポイント上方修正されました[1][2]。

USD/JPY（ドル円）については、7月29-30日のFOMCでは政策金利のフェデラルファンド（FF）レートが4.25-4.50％の据え置きとなりました。米連邦準備制度理事会（FRB）内には9月の利下げを主張する声も出ており（反対票2票）、これがドル円相場に一定の下押し圧力をかける可能性があります。ただし、利下げ前の据え置き姿勢や米中関税緩和などのポジティブ要因でドルの急落は抑えられている状況です[4]。
