世界経済のニュース（2025年4月23日）

2025年4月23日時点で、世界経済は大きな変動を見せています。国際通貨基金（IMF）は、2025年の世界の実質経済成長率を2.8％と予測しました。これは前回の予測から0.5ポイント引き下げられており、主にトランプ氏の高関税政策が影響しています[1][2]。

また、アメリカのトランプ大統領は、対中国向けの関税引き下げに言及し、FRB議長に対する退任要求も撤回しました[3]。

顕著な株価の増減

  • 日経平均株価: 4月23日には、前日比で666.29円上昇し、34,869.98円で推移しました。これは、米国株式市場の大幅反発や円安の影響を受けたものです[1]。

USD/JPY分析

現時点で、米国の経済政策や関税問題が円安に影響を与えており、USD/JPYのレートはこのようなマクロ的な要因に左右されています。円安は輸出関連株を支えると同時に、円高になると輸出企業に悪影響を及ぼす可能性があります。

今後の重要な経済イベント

  • IMFの政策見直し: IMFは全球的な経済政策の不確実性に警戒を続けており、今後の政策見直しが注目されています。
  • トランプ関税問題の進展: トランプ関税を巡る動きが今後も世界経済に大きな影響を与え続けることが予想されます。
  • FRBの金融政策決定: 米連邦準備制度理事会（FRB）の金利決定や経済見通しも世界経済に影響を与えます。
