2. LNG（液化天然ガス）市場

  • 価格動向
    アジアのJKM価格と欧州のTTF価格はともに冬季の需要控えで高値圏を維持。TTFは欧州の冬季備蓄状況が安定傾向で一部圧迫要因だが、供給不安が価格を支えている。

  • 変動要因
    欧州は冬季備蓄が充実しつつも、ロシアからの供給制限リスクが残存。米国のLNG輸出施設は稼働率が高く、需給に大きく影響。冬季需要は依然強い予想であり、これが価格下支え要因。

  • 見通し
    供給側の堅調な輸出態勢と冬の強い需要によって価格は高止まりする見込み。ただし、欧州備蓄状況と地政学的リスクの変動が鍵となる。

3. 電力市場

  • 価格動向

  • 変動要因
    再生可能エネルギーの供給量増加が一部地域で価格抑制要因に。天候による需要変動やLNG・石炭など発電燃料の価格が電力価格に大きく影響。

  • 見通し
    冬季に向けて需要増加と燃料価格の変動が電力市場を左右。再生可能エネルギーの増強状況も価格形成に影響を与え続ける。
