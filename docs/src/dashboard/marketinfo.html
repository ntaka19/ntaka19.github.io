  • 米消費者物価指数（CPI）低下: 米国労働省は3月のCPIが前年同月比で2.4%上昇と発表し、前月の2.8%から低下した。これにより、市場予想を下回る伸びとなった[1]。
  • トランプ関税政策: トランプ大統領が中国向けの関税引き上げを停止する大統領令に署名。同時に関税を引き上げる計画も発表された[1][3]。
  • 金価格史上最高: 米中貿易戦争激化懸念から金価格が史上最高になったと報告されている[5]。

USD/JPYの分析
