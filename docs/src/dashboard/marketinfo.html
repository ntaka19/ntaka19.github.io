3. 電力市場

  • 価格動向: 日本のJEPX、欧州、北米いずれも季節変動により需給が敏感だが、再生可能エネルギーの増加により価格変動は複雑化。日本ではLNGの燃料価格上昇が電力単価に影響し、欧州・北米では気温次第で需要ピークが変動。
