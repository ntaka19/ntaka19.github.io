USD/JPY（ドル円）については、7月29日のニューヨーク市場で一時148円81銭まで上昇した後、148円33銭まで反落して取引を終えました。これは、IMFが2025年の世界および米国経済成長見通しを上方修正し、米国の対前渡商品貿易赤字が予想外に縮小したことからドルが一時強含んだためです。しかし、その後住宅価格指数の伸びが予想を下回り、さらに7年債入札の好調に伴う米国債価格上昇で長期金利低下がドル買いの動きを後退させました。このためドル円は上下変動の後、やや押し戻される形となりました[3][1]。
