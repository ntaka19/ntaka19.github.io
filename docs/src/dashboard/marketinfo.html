世界経済のニュース

2025年5月13日現在、米中間で関税の大幅な引き下げが合意されました。これにより、米中の貿易戦争が事実上収束します[1][2][3]。この合意は、世界経済への楽観的な影響をもたらしており、米国や日本の株式市場が大幅に上昇しました[1][3]。

株価の増減

  • 日経平均株価: 5月13日時点で3万8183円で取引され、3月高値を更新しました。これは前日比539円の増加です[3]。
  • ダウ平均株価: 米国の株式市場も大幅に反発し、上げ幅は1100ドルを超えました[3]。
  • ハイテク株: 米国ではハイテク株にも資金が戻り、リスク選好の状態が進んでいます[3]。

USD/JPYの分析

ドル円は1ドル＝148円台まで上昇し、円安が進んでいます。これは、米中の貿易戦争が収束したことによる投資家のリスク選好の向上が影響しています[3]。このドル高は、米国の経済が安定向きにあり、世界経済への不安が減少していることを反映しています。
