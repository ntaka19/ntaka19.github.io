2025年5月13日時点の世界経済のニュースは、米中経済貿易における重要な動きが注目されています。中国商務部はスイス・ジュネーブでの米中ハイレベル経済貿易会談の共同声明を発表し、双方が課していた関税措置の一部緩和に合意しました。具体的には、中国は追加関税34％のうち24％分を90日間停止し、残りの関税は継続する一方、他の追加関税91％は廃止するとしました。米国側の措置も同様に緩和されることが予定されています。この合意は米中関係にとって重要な一歩と評価され、協議は今後も継続される予定です[1][2]。

この米中関税合意を受けて、株価は大きく上昇しました。特に2025年5月13日の日経平均株価は約1か月半ぶりに3万8000円台を回復し、一時800円を超える上げ幅となりました。アメリカのダウ平均株価も前日12日に1100ドル以上の大幅上昇を記録し、世界的に景気後退への警戒感が和らいだ影響が見られます。日本では半導体関連株を中心に買いが入り、市場は活発化しました。ただし、関税引き下げには期限があり、米中および日米間の今後の貿易交渉の動向には依然として不確実性が残っています[3][4]。

USD/JPY（米ドル/日本円）については、米中関税緩和を受けて世界的なリスクオンの動きが活発化していることから、円安ドル高圧力が強まる傾向にあります。投資家のリスク志向回復によりドルが買われやすく、USD/JPYは上昇基調となっていますが、米中・日米の貿易協議の進展やアメリカの金融政策、日銀の金融緩和政策の動向も影響するため、為替市場は引き続き変動要因が多い状況です。

今後の重要な経済イベントとしては、

  • 米中の貿易交渉の継続と関連する関税措置の期限管理
  • 日米間の貿易交渉、特に自動車産業を巡る動向
  • 米国の金融政策決定（FRBの金利政策など）
  • 日本銀行の金融政策および金融緩和の動向
  • 4月の倒産件数の発表など国内の経済指標

が挙げられます。これらのイベントが市場に与える影響は大きく、特に米中・日米の貿易交渉の進展は世界経済の安定に直結するため、注視する必要があります[1][2][4]。
