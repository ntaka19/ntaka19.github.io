4-3. 電力市場

価格動向

日本（JEPX）

  • 燃料費連動と需給ひっ迫で大きく動く
  • 夏冬のピーク需要時は、再エネ出力低下や気温変動の影響を受けやすい

欧州

  • ガス価格（TTF）と再エネ発電量の影響が大きい
  • 風力発電の弱い日は電力価格が急騰しやすい

北米

  • 天然ガス価格と地域別の送電制約、気温需要が主因
  • 冷房・暖房需要でスポット価格が変動しやすい

主要な変動要因

1) 再生可能エネルギーの供給量

  • 風力・太陽光の発電量が多いと卸電力価格は下がりやすい
  • 出力が落ちると火力依存が高まり、価格上昇圧力

2) 気温変動による需要

  • 猛暑・厳冬は需要増で価格上昇
  • 平年並みの気温なら安定しやすい

3) 発電燃料価格

  • LNG高騰 → 日本・欧州の電力価格上昇
  • 石炭高 → ベースロードコストが上昇

今後の見通し

  • 日本は、夏季需要ピーク前の在庫・供給安定性が焦点
  • 欧州は、ガス在庫と風況
