3. 電力市場

  • 価格動向: 日本（JEPX）、欧州、北米ともに再生可能エネルギーの増加が全体の供給構成を変化させつつも、猛暑や冷夏による季節的需要変動で価格が変動。燃料価格上昇は電力価格の底上げ要因。
  • 変動要因: 再生可能エネルギー供給の不安定さに加え、LNGや石炭価格の変動が火力発電コストに直結。気温変化による季節需要の増減が電力価格の変動に大きく影響している。
