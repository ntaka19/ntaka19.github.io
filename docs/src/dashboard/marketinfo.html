2. LNG（液化天然ガス）市場

  • 価格動向
    アジアのJKM（日本・韓国・中国を中心とした指標）と欧州のTTFでは、冬季需要の高まりを受け価格が上昇傾向。特に欧州は備蓄レベルが例年より低く、TTF価格はJKMを上回っている状況が続いていると推測される。

  • 米国の輸出施設の生産能力は増加傾向だが、供給制約が依然存在。

3. 電力市場

  • 価格動向
    日本（JEPX）、欧州、北米の電力市場で価格は季節要因（冬季到来前の需要増加）と燃料価格の影響を受けて上昇傾向。再生可能エネルギーの供給量に左右されつつ、LNGや石炭価格上昇が電力コストを押し上げている。

  • 変動要因

  • 再生可能エネルギーの供給増加が価格変動の重要ファクター。
