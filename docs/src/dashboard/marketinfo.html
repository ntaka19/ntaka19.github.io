2. LNG（液化天然ガス）市場

  • 価格動向

3. 電力市場

  • 価格動向
    日本（JEPX）、欧州、北米の主要電力市場ともに夏季のピーク需要により価格が上昇傾向。特に電力需給がタイトな地域では価格高止まりしている。

  • 変動要因
    再生可能エネルギーの供給拡大が価格の一部抑制要因だが、気温の変動による冷房需要増加が価格を押し上げ。発電燃料価格（LNG・石炭）の変動が電力コストに直結しているため注意が必要。
