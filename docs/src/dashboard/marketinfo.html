2025年8月7日時点の世界経済ニュースとして、アメリカは日本を含む69カ国・地域に対する相互関税率を変更し、日本からの輸入品に対する追加関税が15％に引き上げられた。この新関税率は8月7日米国時間午前0時1分以降に通関された貨物に適用されている[1][3]。一方、世界経済成長率は上方修正されており、2025年の米国成長率は1.9％、ユーロ圏は1.0％、中国は4.8％、インドは6.4％と予測されている[2]。
