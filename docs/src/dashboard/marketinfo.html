    欧州ではロシアとポーランド間の軍事緊張が続き、EUは対ロシア制裁強化を表明している。また、中国の8月消費者物価指数（CPI）が前年比-0.4％と3カ月ぶりにマイナスに転じ、景気回復への懸念が強まっている。米国ではCPI発表前にFRBによる利下げ再開がほぼ確実視されている状況で、インフレと景気の方向性が注目されている[1][2]。

  • 株価動向

  • ドル円（USD/JPY）動向

商品市場の動向

1. 原油市場

  • 価格動向
    直近1週間のWTI・ブレント原油価格は全体的に上昇基調を維持しているが、1ヶ月単位で見ると小幅な上昇トレンドにある。これはOPEC+が計画的に減産を継続していることに加え、地政学的リスクの高まりが投資マネーを呼び込んでいるためである。

  • 変動要因

  • OPEC+の減産継続により供給タイト化が進行。
  • 中東、特にホルムズ海峡周辺の軍事的緊張が供給不安を刺激。
  • 米国の石油在庫は横ばいもしくはわずかに減少傾向で、需給バランスは引き締まっている。

  • 世界経済の成長鈍化懸念があるものの、新興国需要は底堅く、全体の需要見通しは控えめながら安定的としている。

  • 今後の見通し
    短期的には地政学リスクとOPEC+の政策次第で上下はあるものの、減産効果が継続し需要も一定の水準であり、価格は底堅く推移すると予想される。

2. LNG市場

  • 価格動向
    アジア向けJKM価格と欧州向けTTF価格は共に季節的な冬季需要に向けてやや上昇基調にある。とくに欧州は備蓄水準が前年よりも良好なため、価格はアジアと比べてやや落ち着いているが、冬季に向けては上昇圧力が強まっている。

  • 変動要因

  • 欧州の備蓄充足率が高く、冬季に供給不足となるリスクは抑えられている。
  • 米国のLNG輸出施設は生産能力を拡大中で供給余力が増している。

  • 冬季の寒波予測とそれに伴う暖房需要が価格を下支えしている。

  • 今後の見通し
    需要増加が見込まれる冬季に向けて価格は全般的に高止まりしやすいが、供給側の生産拡大と備蓄状況により過大な価格高騰は回避される見通し。

3. 電力市場

  • 価格動向
    日本（JEPX）、欧州、北米それぞれで気温の変動により電力需要が季節的に上下している。再生可能エネルギーの供給拡大がある一方、LNGや石炭価格の変動が電力価格に直接影響している。

  • 変動要因

  • 再生可能エネルギー（太陽光・風力）の増加により電力供給が多様化。
  • 気温の上昇・低下で冷暖房需要が変動、価格を揺らす。

  • LNG・石炭燃料コストの高低が直接的に発電コストへ反映。

  • 今後の見通し
    再生可能エネルギーの寄与によりピーク価格は押さえられるものの、燃料価格の高止まりが価格変動のリスクとなる。気象状況に左右されやすい状況は引き続き続く見込み。

4. 貴金属・LME市場

  • 金・銀（貴金属）
  • 価格動向: ドル建て金・銀価格は直近で上下動が見られるが、FRBの利下げ観測による金利低下予想を背景に買い戻しが強まっている。

  • 変動要因: 高金利時代が一服し、ドル安期待が安全資産である金・銀の需要を押し上げ。金融政策の不確実性も価格を支える要因。

  • 銅・アルミなどLME非鉄金属

  • 価格動向: 銅、アルミ価格は中国の景気動向を敏感に反映し、ここ数週間は持ち直し傾向。
  • 変動要因: 中国の製造業活動が若干の回復を示し需給が改善傾向。主要生産国の供給安定が価格を支えている。世界的な製造業の見通し安定化もプラス要因。
