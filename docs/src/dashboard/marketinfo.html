補足説明
- FOMC（連邦公開市場委員会）: 米国の金融政策を決定する会合。金利変更など重要指標発表が市場に大きな影響を与える。

- LME: ロンドン金属取引所。銅やアルミなどの非鉄金属の国際価格の基準。
