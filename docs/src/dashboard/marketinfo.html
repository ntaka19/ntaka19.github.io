用語補足

  • WTI/ブレント原油: 世界の代表的な原油価格指標。WTIは米国産、ブレントは北海産原油を指す。
  • JKM: アジアの液化天然ガス価格指標。
  • TTF: 欧州の天然ガス取引所価格指標。
  • JEPX: 日本卸電力取引市場。
  • LME: ロンドン金属取引所。銅、アルミなど非鉄金属の主取引市場。
  • CPI: 消費者物価指数、インフレ指標として重要。
