今日の天気 (07-18 Sat)

皆さん、おはようございます！7月18日の天気をお伝えします。

今日の天気 東京は一日を通して雲が多めの空模様となるでしょう。特に午前中は、ごく弱い小雨がぱらつく時間帯もありますので、念のため折りたたみ傘があると安心です。午後は雲の合間から日差しが差し込むこともありますが、全体的にはすっきりしない一日となりそうです。 最高気温は27度台後半、最低気温は24度台後半で、蒸し暑さが続くでしょう。

日傘は必要？ はい、日傘はぜひお持ちください。 今日のUVインデックスは最大で7.7と「高い」レベルが予想されています。日照時間は短い予報ですが、雲の切れ間からでも紫外線は降り注ぎます。油断せず、日焼け対策をしっかりと行ってくださいね。
