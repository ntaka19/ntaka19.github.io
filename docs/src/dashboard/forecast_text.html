日傘（パラソル）について

  • UV指数7 は「中～高」レベルで、長時間外出すると紫外線による肌ダメージや熱中症のリスクがあるため、日傘を持参するとよいでしょう。
  • 晴天で風が弱めなので、風を避けて涼しく過ごすためにもパラソルは実用的です。
  • ただし、紫外線対策としては日傘に加えて、日焼け止めと帽子の着用をおすすめします。

結論
午後に外出予定がある場合は「日傘を持って行った方が安心」です。晴れの日の活動でも紫外線対策をしっかり行うと良いでしょう。
