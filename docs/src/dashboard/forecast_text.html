日傘を持っていくほうが賢いか？
- UV 6は日光の強さが高いので、昼過ぎ（12：00〜16：00）に屋外で長時間いる場合は日傘をおすすめします。
- それ以外、短時間の散歩や出張であれば必ずしも必須ではありませんが、日差しが強いときは短い枝付きの傘でも十分にサンプロテクションになります。
- さらに紫外線対策として日焼け止め（SPF 30以上）とサングラスの併用も有効です。
