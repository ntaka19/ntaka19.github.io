今日の天気 (07-01 Wed)

皆さん、こんにちは！7月1日、水曜日の東京のお天気をお伝えします。

今日の東京は、一日を通して雲が多めの空模様となるでしょう。ただ、時折晴れ間がのぞき、ムシムシとした夏らしい一日となりそうです。予想最高気温は27度、最低気温は22度と、この時期らしい気温となるでしょう。雨の心配はありません。

気になる紫外線情報です。本日のUVインデックスの最大値は「7.6」と、非常に強いレベルとなっています。日照時間は2時間20分あまりと短めですが、雲の切れ間からの日差しは大変強いでしょう。

日傘は必要？

はい、日傘はぜひお持ちください。
