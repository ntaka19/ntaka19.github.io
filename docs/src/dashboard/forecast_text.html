今日の天気 (07-13 Mon)

7月13日月曜日、東京の天気予報をお伝えします！

この日の東京は、一日を通して雨模様となりそうです。特に午前中と夕方から夜にかけては、しとしととした雨や霧雨が降る時間帯が多くなるでしょう。気温は、朝晩で24℃台、日中も最高で28℃台と、この時期としてはやや肌寒い一日となりそうです。

さて、日傘についてですが、この日は日照時間が全くなく、太陽の光が届くことはありません。最高紫外線指数も3.9（中程度）とされていますが、直射日光がないため、日差し対策としての「日傘」は特に必要ありません。

ただし、一日を通して雨が降る予報ですので、雨を防ぐための傘は必ずお持ちくださいね！
