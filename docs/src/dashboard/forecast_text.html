紫外線5以上は「中〜やや高い」に該当し、日焼け防止が必要です。特に昼間（10–15 時）は紫外線がピークになるため、直射日光を浴びるのは避けるか、少なくとも30 分以上屋外にいる予定なら日傘を持っていくと安心です。

日傘は必要？

  • はい。
  • 午前10時以降の直射日光が強いうちに外出する予定があるなら、日傘（または帽子＋サングラス）を装備すると肌のダメージや熱中症リスクを減らせます。

  • 軽い曇りが出る時間帯（20時前後）では紫外線は低下しますが、午後に再び曇りが増えると紫外線は緩和されます。

  • 特に長時間外にいる場合（2 時間以上）は必ず持参をおすすめします。短時間（15 分以内）であれば、日傘は必須ではありませんが、日焼け止めを併用するとさらに安全です。
