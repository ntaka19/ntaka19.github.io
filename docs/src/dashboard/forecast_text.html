日傘の必要性
UV index 6 〜 7は「高」カテゴリーです。長時間外出する場合は紫外線対策が必須。
 風が強い日や午前・午後の時間帯に日傘は有効です。
* さらに、熱中症対策として日陰と軽装の併用をおすすめします。

まとめ

晴れのち薄曇り、最高気温 ≈ 32 ℃。紫外線は高い（UV 6.8）ので、日傘を持って外出する方が安全です。
