ポイント
- 午前から午後にかけては「雨・曇り」、夕方以降は「晴れ・曇り」。
- 低めのUVでも軽度の日焼けになる可能性があるので、外出時は日焼け止めを使用。
- 折りたたみ傘を手元に置いておけば、突発的な雨でも対応可能です。
