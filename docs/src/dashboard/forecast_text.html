日傘（サンブレイ）について
- 雲が多い時間帯でも紫外線は中等度。
- 日中（10:00 〜 16:00）に屋外で長時間過ごすなら、光線対策として小さめの日傘または帽子を持っていくと安心です。
- ひた隅で日帰り観光など軽く持ち歩けるくらいの長さがあれば十分です。

まとめ
天候は快晴からくすみのある曇りで、雨の心配はないので日傘を必須にする必要はありません。ただし、日中に紫外線対策をしたい場合は、軽い日傘を持参すると良いでしょう。
