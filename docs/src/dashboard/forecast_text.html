日傘（雨傘）について

  • 雨：午後は降雨なし。朝の5時～9時頃までに軽い霧雨が予想されるので、朝出掛けるときは小さめの雨傘を持つと安心です。

総じて、軽い雨傘は朝の時間帯に持参すれば安心ですが、午後からは晴天で日傘は不要です。温度も比較的にじやかなので、外出は快適に過ごせるでしょう。お散歩や外出の際は、朝の霧雨に備えて小型の傘をもう一つ持っておくといいかもしれません。
